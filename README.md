# Video-compressor-bot-v1


প্রথমে আপনার টার্মাক্স টার্মিনালে গিয়ে নিচের কমান্ডটি দিয়ে নতুন বটের ফাইলটি স্ক্রিনে ওপেন করুন:

Bash
nano main.py




টার্মাক্সে নিচের কমান্ডটি দিয়ে ফাইল ওপেন করুন:

Bash
nano requirements.txt

এবার বটের সাথে টেলিগ্রামের সংযোগের জন্য নিচের কমান্ডটি দিন:

Bash
nano .env

Bash
# প্রয়োজনীয় প্যাকেজ ইনস্টল করা
pip install -r requirements.txt

# বট রান করা
python main.py
